Cara untuk memasukan tulisan tangan mu sendiri

  • Untuk menggunakan tulisan tangan mu sendiri, kamu harus membuat tulisan tanganmu menjadi sebuah font.
  • Untuk membuatnya bisa melalui Website berikut Calligraphr atau juga bisa menggunakan jasa profesional untuk membuat font.
  • Saat sudah mendapatkan font mu sendiri dengan format .ttf contoh : namakmu_font.ttf upload font tersebut di bagian Upload Tulisan Tangan Sendiri pada bagian Opsi Tulisan Tangan
